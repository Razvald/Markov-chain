# Генератор текстов на основе цепей Маркова

## Описание проекта

Проект представляет собой **генератор псевдореалистичных текстов** на русском языке с использованием **цепей Маркова**.
Алгоритм обучается на корпусе текстов, строит статистику последовательностей слов и генерирует новые тексты, сохраняя стилистические особенности исходного материала.

Проект покрывает ключевые этапы:

* Токенизация и очистка текста
* Построение модели переходов слов (n-граммы, здесь n=2)
* Генерация текста
* Простая визуальная оболочка (UI) для работы с текстами

### Использование

#### 1. Запуск приложения

```bash
pip install streamlit
streamlit run app.py
```

Откроется веб-интерфейс в браузере.

#### 2. Работа с приложением

* Загрузить один или несколько `.txt` файлов
* Выбрать длину текста
* Нажать **Сгенерировать**
* При необходимости сохранить результат в файл

---

## Логика работы

1. **Токенизация**

   * Текст разбивается на слова, знаки препинания удаляются.
   * Определяются слова, с которых начинаются предложения.

2. **Построение модели**

   * Используются **2-gram цепи Маркова**: ключ — пара последовательных слов `(w1, w2)`, значение — список слов, которые встречаются после этой пары.

3. **Генерация текста**

   * Выбирается **начальная пара слов** случайно (с учётом частоты появления в начале предложений для разнообразия).
   * Далее слова добавляются одно за другим на основе статистики цепей.
